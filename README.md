# Personal Portfolio Website

This is a responsive multi-page portfolio website built using HTML, CSS, and JavaScript.

## Features
- Multi-page navigation (Home, CV, Schedule, Contact)
- Responsive design for mobile and desktop
- Interactive contact form with JavaScript
- Styled components using modern CSS
- Organized layout and clean UI structure

## Technologies Used
- HTML
- CSS
- JavaScript

## Description
This project demonstrates front-end development skills, including layout design, responsive styling, and basic interactivity. It simulates a personal portfolio website with multiple functional pages.
